# valorant-picker
Valorant Picker — это Android-приложение, созданное для того, чтобы помогать игрокам Valorant делать более осознанный выбор агентов.

Пользователь выбирает:

-карту, на которой будет проходить матч;

-агентов, которых уже выбрали его тиммейты.

На основе этих данных приложение определяет наиболее подходящего агента для игрока.
Проект вдохновлён такими инструментами, как DotaPicker, но адаптирован специально под Valorant.

Используемые технологии(на данный момент):

-Kotlin

-Jetpack Compose

-архитектура MVVM

-StateFlow

-Navigation Compose

Основная цель проекта — как практическая (помощь игрокам), так и учебная: изучение современных подходов к Android-разработке.
