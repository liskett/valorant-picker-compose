# Valorant Picker

Valorant Picker — это Android-приложение, которое помогает игрокам Valorant выбрать агента перед матчем, исходя из карты и уже выбранных агентов в команде.

Идея проекта появилась из личного опыта: часто в матчах приходится выбирать агента наугад, не учитывая общую картину пика. Этот проект — попытка сделать процесс выбора чуть более осознанным. Вдохновлялся инструментами вроде **DotaPicker**, но адаптировал логику именно под Valorant.

---

## Цели проекта

У проекта две основные цели:

* **Практическая** — помочь игрокам быстрее и удобнее определиться с агентом
* **Учебная** — на практике разобраться в современных подходах к Android-разработке

Проект используется как pet-проект и площадка для экспериментов с архитектурой, состояниями и UI.

---

## Что умеет приложение

Пользователь может:

* выбрать карту, на которой будет проходить матч;
* отметить агентов, которых уже выбрали тиммейты.

На основе этих данных приложение:

* анализирует текущий пик команды;
* предлагает агента, который лучше всего подойдёт в данной ситуации;
* отображает результат в простом и понятном интерфейсе.

---

## Архитектура и подходы

Проект построен на **MVVM** — без усложнений, но с чётким разделением ответственности:

* **UI** — Jetpack Compose, отвечает только за отображение состояния
* **ViewModel** — хранит состояние экрана и бизнес-логику
* **StateFlow** — используется для управления состоянием и обновления UI

Такой подход позволяет легко поддерживать код, расширять функциональность и в дальнейшем без проблем добавлять тесты.

---

## Используемые технологии

* **Kotlin** — основной язык разработки
* **Jetpack Compose** — декларативный UI
* **MVVM** — архитектура приложения
* **StateFlow** — управление состоянием
* **Navigation Compose** — навигация между экранами
